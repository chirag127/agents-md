## Continue-specific quirks

- **`.continuerules`** (legacy single file) and **`.continue/rules/*.md`** (current directory form) both work. Global rules go in `~/.continue/global-rules.md`.
- **Custom slash commands:** Continue lets you define `/cmd` shortcuts in `config.yaml`; keep this stub small and put commands in the user's config.
